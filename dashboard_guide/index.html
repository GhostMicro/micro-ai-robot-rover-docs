    🤖 Ghost Micro - Commander Dashboard Guide

    Welcome to the Ghost Micro Commander Interface (v3.0). This guide explains how to operate the dual-control system using the PC Dashboard and the interactive OLED UI.

    1. Manual Keyboard Control (8 Keys Total)

    You have full control using 8 active keys:

    Movement (Drive & Cancel Action)

    • [W]: Move Forward
    • [S]: Move Backward
    • [A]: Turn Left
    • [D]: Turn Right (Pressing any WASD key immediately cancels any active Action/Dance)

    Action Triggers (1-4)

    • [1]: Square Step (Walks in a box shape)
    • [2]: Zig Zag (Moves forward in a zigzag pattern)
    • [3]: Shake (Shakes left and right in place)
    • [4]: Slow Spiral (Drives in a widening circle)

    2. Dual-Source Logic

    The system intelligently switches between inputs: - PC Priority: As long as the Dashboard is open, it takes priority over the physical Joystick. - Joystick Fallback: If the PC is inactive for >2 seconds, the physical hardware joystick is re-enabled.

    🚀 Advanced Features

    ⚡ Speed Control

    Use the SPEED CONTROL slider to adjust the driving intensity: - Low Speed (20-50): Ideal for precise indoor maneuvers. - High Speed (100-127): Best for outdoor straight-line speed.

    🎭 Special Features

    Pre-programmed maneuvers accessible via dashboard buttons or keys: - SPIN 360: Commanding the robot to perform a continuous rotation. - DANCE: Triggers a sequence of pre-defined movements.

    🎨 Visual Customization

    • Toggle Theme: Switch the UI between Neon Green and Cyber Blue to suit your style.

    📺 OLED UI Feedback (Transmitter)

    The OLED display on your transmitter provides real-time telemetry:

    Color Zones

    • YELLOW (Top 16px): Displays "GHOST MICRO" and the active source ([JOY], [PC], or [.]).
    • BLUE (Bottom 48px): Displays the joystick visualizer (Crosshair), X/Y coordinates, and RF connection status.

    Status Indicators

    • [JOY]: Control is currently coming from the physical Nano joystick.
    • [PC]: Control is currently coming from this Dashboard via USB.
    • [.]: Idle / No connection active.

    🛠️ Setup & Requirements

    1. Connection: Connect the ESP32 Transmitter to your PC via USB.
    2. Environment: Ensure customtkinter and pyserial are installed.
    3. Run: bash python3 dashboard.py
