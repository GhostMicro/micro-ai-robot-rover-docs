# ปรัชญาและแรงบันดาลใจ (Philosophy & Inspiration)

## จุดเริ่มต้น (The Origin)
โครงการ **Micro AI Robot Rover** เกิดขึ้นจากความต้องการสร้างหุ่นยนต์ที่ **"ฉลาด" (Smart)** แต่ต้อง **"ยืดหยุ่น" (Flexible)** พอที่จะปรับเปลี่ยนรูปร่างหน้าตาได้ตามภารกิจ โดยไม่ต้องรื้อระบบสมองใหม่ทุกครั้ง

เราได้รับแรงบันดาลใจจากแนวคิด **Cyberpunk** ที่ผสมผสานเทคโนโลยีล้ำสมัยเข้ากับฮาร์ดแวร์ที่ดูดิบและใช้งานจริง (Industrial Functionalism)

## ปรัชญาการออกแบบ (Design Philosophy)

### 1. The Universal Brain (สมองสากล)
หัวใจสำคัญของโปรเจกต์นี้คือ **"สมองต้องแยกออกจากร่างกาย"**
- **สมอง (Brain Box):** กล่องควบคุมที่มี ESP32 และ AI ทำหน้าที่ตัดสินใจและสื่อสาร
- **ร่างกาย (Body Chassis):** โครงหุ่นยนต์ มอเตอร์ และล้อ ซึ่งเปลี่ยนไปตามสภาพพื้นที่ (ล้อตีนตะขาบ, ล้อ Mecanum, หรือแม้แต่โดรน)

เมื่อเราถอด "สมอง" ออกจากหุ่นตัวหนึ่ง ไปเสียบใส่อีกตัวหนึ่ง หุ่นตัวใหม่จะต้อง "มีชีวิต" ทันทีโดยไม่ต้องเขียนโค้ดใหม่ เพียงแค่เปลี่ยนการตั้งค่าเล็กน้อย

### 2. Offline-First & Privacy
หุ่นยนต์ต้องทำงานได้สมบูรณ์แบบแม้ไม่มีอินเทอร์เน็ต (No Cloud Dependency) ข้อมูลทุกอย่างประมวลผลบนตัวรถ (Edge Computing) และ Web Interface สำหรับควบคุมต้องรันจาก Server บนตัวรถเอง เพื่อความปลอดภัยและความเป็นส่วนตัวสูงสุด

### 3. Modular & Scalable
การออกแบบต้องเป็นโมดูล (Modular) เชื่อมต่อกันด้วยพอร์ตมาตรฐาน อะไหล่ต้องหาได้ง่าย (Arduino/ESP32 ecosystem) และสามารถอัพเกรดเฉพาะส่วนได้ เช่น อัพเกรดกล้อง อัพเกรดมอเตอร์ โดยไม่กระทบส่วนอื่น
