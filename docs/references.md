# แหล่งอ้างอิงและผู้สนับสนุน (References & Sponsors)

## แหล่งข้อมูลอ้างอิง (References)

### Libraries ที่ใช้
*   **ESP32Servo:** สำหรับควบคุมเซอร์โวมอเตอร์บน ESP32
*   **NewPing:** ไลบรารีมาตรฐานสำหรับการอ่านค่า Ultrasonic Sensor ที่แม่นยำและไม่บล็อกการทำงาน
*   **MPU6050_tockn:** สำหรับอ่านค่า Gyroscope และ Accelerometer แบบง่าย
*   **Adafruit_BMP280:** สำหรับอ่านค่าเซนเซอร์ความดันอากาศ HW-611
