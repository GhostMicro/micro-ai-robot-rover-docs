# คู่มือการติดตั้งและใช้งาน Fritzing (Circuit Design)

สำหรับการออกแบบวงจรหุ่นยนต์ต้นแบบ (Prototype) เราแนะนำให้ใช้ **Fritzing** เพราะมีมุมมอง "Breadboard" ที่เสมือนจริงและใช้งานง่าย

---

## 1. วิธีการติดตั้ง (Installation)

สำหรับการใช้งานบน **Ubuntu 24.04** แนะนำให้ติดตั้งผ่าน **Flatpak** เพื่อความเสถียรและได้เวอร์ชันล่าสุด (ซึ่งได้ดำเนินการติดตั้งให้เรียบร้อยแล้ว):

```bash
# คำสั่งติดตั้ง (หากต้องการติดตั้งใหม่)
flatpak install flathub org.fritzing.Fritzing
```

---

## 2. วิธีการเปิดโปรแกรม (Launch)

คุณสามารถเปิดโปรแกรมได้ผ่านเมนู Application ของระบบ หรือใช้คำสั่งใน Terminal:

```bash
flatpak run org.fritzing.Fritzing
```

---

## 3. ฟีเจอร์ที่สำคัญสำหรับการทำหุ่นยนต์

*   **Breadboard View:** ใช้สำหรับวางแผนการต่อสายไฟ เหมือนการจิ้มลงบนบอร์ดจริง ช่วยลดความผิดพลาดก่อนการบัดกรี
*   **Schematic View:** ใช้สำหรับดูผังวงจรหลักตามหลักการไฟฟ้า (สำหรับส่งต่อในคู่มือเทคนิค)
*   **PCB View:** สำหรับผู้ที่ต้องการขยายผลไปทำแผงวงจรพิมพ์ในอนาคต

---
